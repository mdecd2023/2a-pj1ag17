創網站跟上傳步驟

步驟簡述

先下載Filezilla portable 設定好站台(比較好用

git config --global http.proxy http://p42.cycu.org:3128 

  git config --global user.name "name" 

  git config --global user.email "email" 

打完會產生.gitconfig

git clone --recurse-submodules git@demo:mdecad2022/site-github_帳號.git s 

  會產生site-學號

上傳指令

source acp "提交說明字串" 

 

下載server.py和config 放入倉儲(server.py要改成自己的埠號)

cd s, cd cmsimde, git pull origin master 

  更新版本(如果有改過密碼要把舊的config刪掉)

 

python3 server.py 

  執行server

如果server不能執行

ps axo pid.comm.user

可以查詢到以執行server

kill -9 111919 

可以殺掉自己的server
